This is the code for the WebListers take home quiz.

## Quick Start

```bash
# Install dependencies
npm install

#######################################################################################
#######################################################################################
# DEVELOPMENT w/ mock data & json-server
# Open a command terminal and from the root folder run 
npm run server

# And from another terminal window, or IDE terminal
# Serve the app on localhost:4200
# Make sure `production: false` is set in both environment & environment.prod.ts files
ng serve or npm run start
#######################################################################################
#######################################################################################
# PRODUCTION: locally hosted front end, back end produciton
# Open a command terminal and from the root folder run 
# Make sure `production: true` is set in both environment & environment.prod.ts files
ng serve --prod or npm run serve:prod

# Build for production but has to be placed on a weblister server
ng build --prod or npm run build
```
http://localhost:4200/list-user
